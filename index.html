Buscador de CEP

O projeto consiste na criação de um buscador de CEP (Código de Endereçamento Postal) utilizando tecnologias web modernas. O objetivo principal é permitir que os usuários insiram um CEP e obtenham informações detalhadas sobre o endereço correspondente de forma rápida e intuitiva.

Pagina de Login

O projeto consiste na criação de uma página de login utilizando tecnologias web, sem a necessidade de integração com um banco de dados. O objetivo principal é fornecer uma interface simples para os usuários acessarem a aplicação com credenciais pré-definidas no código.

Relogio / Contagem Regressiva

O projeto consiste na criação de um relógio digital que exibe a hora atual em tempo real e realiza uma contagem regressiva até o final do ano corrente utilizando tecnologias web. A aplicação permitirá aos usuários visualizarem o tempo restante até a virada do ano de forma interativa e informativa.
